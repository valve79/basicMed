Компания CorePartners Inc. предлагает систему корпоративного видеохостинга, которая обеспечивает профессиональные возможности для воспроизведения качественного видео

  • РАСШИРЕННЫЙ ПОИСК ПО КОНТЕНТУ
    • Поисковая система сочетает возможности поиска по содержанию видеофрагментов с традиционным поиском по ключевым словам из названия контента.
  • ПЕРСОНАЛИЗАЦИЯ И ЛИЧНЫЙ КАБИНЕТ
    • Core ACHIEVE поддерживает гибкую систему персонализации личного кабинета на основе распределения ролей. Атрибутивный подход позволяет ограничивать время нахождения контента в открытом доступе.
  • АНАЛИТИКА И ФИНАНСОВЫЕ ПОКАЗАТЕЛИ
    • Core ACHIEVE предлагает большое число предустановленных отчётов и позволяет осуществлять гибкую настройку данных.
  • ПЛАТЕЖИ И БИЛЛИНГ
    • Core ACHIEVE осуществляет поддержку моментальной оплаты с помощью банковских карт (Visa, MasterCard), сервисов WebMoney и PayPal.

Для кого эта система?

Решение предназначено для университетов, колледжей и других образовательных учреждений. Система обеспечивает максимально гибкую реализацию и легко масштабируется под нужды Заказчика.

Как это работает?

Сервис видеохостинга базируется на технологии потокового видео, которая позволяет осуществлять загрузку и просмотр пользовательского видео больших размеров на максимальной скорости передачи данных.
